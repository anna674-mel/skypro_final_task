Git

  • git clone - склонировать удалённый репозиторий.
  • git remote add origin - связать удалённый и локальный репозитории.
  • git pull - подтянуть изменения.
  • git status - посмотреть статус файлов.
  • git add - добавить файлы в индекс.
  • git commit -m - сделать коммит.
  • git log - посмотреть историю коммитов.
  • git push - запушить изменения.
